Hospital Bulk Processing API

This service provides APIs to validate and bulk-create hospital records by integrating with the Hospital Directory API.

1. CSV Validation API

POST /hospitals/bulk/validate

Description

Validates the uploaded CSV file before processing. No hospital records are created using this endpoint.

Request

Content-Type: multipart/form-data

Field Type Required Description
file CSV File Yes CSV file containing hospital data

CSV Format

name,address,phone
General Hospital,123 Main St,555-1234
City Care,456 Park Ave,

Success Response

{
  "valid": true,
  "total_hospitals": 2,
  "preview": [
    {
      "name": "General Hospital",
      "address": "123 Main St",
      "phone": "555-1234"
    }
  ]
}

2. Bulk Create Hospitals API

POST /hospitals/bulk

Description

Validates a CSV file and process it to create multiple hospital records using the Hospital Directory API. A unique batch ID is generated for each request.

Request

Content-Type: multipart/form-data

Field Type Required Description
file CSV File Yes CSV file containing hospital records

Success Response

{
  "batch_id": "uuid",
  "total_hospitals": 2,
  "processed_hospitals": 2,
  "failed_hospitals": 0,
  "processing_time_seconds": 3,
  "batch_activated": true,
  "hospitals": [
    {
      "row": 1,
      "hospital_id": 101,
      "name": "General Hospital",
      "status": "created_and_activated"
    }
  ]
}
Note: Batch activation is performed only if all hospital records are created successfully.
